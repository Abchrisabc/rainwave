We highly encourage you use Rainwave as background music for your streams! All our music is volume-normalized and ad-free, perfect for background as part of a stream!

Copyright/Content ID, and "Safe For Work"

The tl;dr is use OverClocked ReMix/OCR Radio. Your stream is important, though, so please review the following and select a station carefully. Lyrical content is noted, as some streamers will not appreciate having a song with lyrics competing on stream with their own voice for user's attention, even if the song is turned down to be in the background.

  • All has some lyrical content, and is "rated G"/"safe for work". (no swears, no racial slurs, no hentai game content or album art, etc) You stand a strong risk of being Content IDed when using this station.
  • Game has very little lyrical content, and is rated G/safe for work. You will almost certainly be Content IDed.
  • OverClocked ReMix has no copyrighted content, has some lyrical content, and is rated PG-13 with the very rare R. We defer to OverClocked ReMix's judgement, which in rare cases produces content such as Niggaz 4 Life. More importantly, though, you will not be Content IDed with OCR.
  • Covers has a lot of lyrical content, and is rated PG-13 for swearing and themes from some remixes with lyrics. You have a very small chance of being Content IDed.
  • Chiptune has no lyrical content, and is rated G. You have a small chance of being Content IDed.

How To Use in OBS

Generate a URL from the options below. Use the CLR Browser Plugin for OBS or the Browser Plugin for OBS Studio/Multiplatform, and put the generated URL into the browser plugin's configuration.

To put the audio on your stream, listen to Rainwave as you normally would and capture regularly through desktop audio in OBS. You can use our website running in the background, or use your media player of choice by downloading M3Us using the audio player menu on our site by hovering over the play icon. There will be no need to adjust the volume once started, Rainwave is kept at a consistent volume through per-song ReplayGain normalization.

Build Your Widget

Option Value Description
station Which Rainwave station to use. OverClocked ReMix is highly recommended as you have zero risk of being Content IDed.
layout Where the album art goes relative to the song details.
art_size Album art size in the widget.
text_align Alignment of text.
show_artist Show artist or not in the widget's text.
color CSS color rule for font color. (e.g. "white"/"#FFF", "black"/"#000", or use a color picker)
background_color CSS rule for the background color for the song box. Leave blank for transparent. You may need to add your own background in OBS if using transparent.
box_shadow CSS box-shadow/glow rule for the container. (use this page for help - take what comes after "box-shadow:" and paste into this setting) Leave blank for no shadow/glow.
font_size CSS rule for font size. Can use pixels (e.g. "14px") or points. (e.g. "14pt") Leave blank for Rainwave defaults.
font_family CSS rule for what font to use. Font names with spaces must be enclosed with single quotes, e.g. 'Helvetica Neue'. Leave blank for Rainwave defaults.
text_stroke CSS color rule for text-stroke. See here for details. (only previewable in Chrome/Safari) Leave blank for no text-stroke.
text_shadow CSS text-shadow rule. See here for details. Leave blank for no shadow.
width CSS rule for width of the widget (including album art, but excluding padding) before text gets clipped.
padding CSS rule for the space between the edge of the widget and the art/text.
art_shadow CSS box-shadow/glow rule for the art. (use this page for help - take what comes after "box-shadow:" and paste into this setting) Leave blank for no shadow/glow.
ad How many songs to play before showing Rainwave's logo as the album art for a brief moment. Set to 0 to never show an ad. Ads last 2-5 seconds depending on the setting, and are Rainwave's logo with our URL displayed in the same style as song information. (please be kind, we run this service for free :) )
ad_message Banner message for the ad. Enter in the language(s) of your stream, or leave at default for "Music by:".
